Visible Bone FBX Exporter

本アドオンは、BlenderからのFBX出力時に、「表示されているボーン（Bone Collectionの表示設定）」のみを抽出して出力するツールです。🐾
VRChat向けアバターの最適化や、Unityへの軽量なデータ移行に役立ちます。

🎮 使い方 (Usage)

インストール

  • Blenderの 編集 → プリファレンス → アドオン → インストール から .py ファイルを選択して有効化。

サイドバーを開く

  • 3Dビューポート上で Nキー を押し、「FBX出力」タブを開きます。

設定の確認

  • 「表示ボーンのみ抽出する」にチェックが入っていることを確認します。

エクスポート実行

  • エクスポート画面を開く ボタンをクリックし、通常のFBXエクスポートと同様に保存先を指定して完了。

⚠️ 注意事項・仕様

【非破壊処理】

通常は元の状態に復元されますが、異常終了時に残骸が残る場合があります。

  • 親子関係: 親ボーンが非表示で子ボーンが表示されている場合、子ボーンは保持されますが親子関係が解除される場合があります。
  • FBX構造の変化: ボーン数が変わるため、Unity側でのウェイトや動作を必ず確認してください。

🛠️ トラブルシューティング

残骸データが残ってしまった場合

万が一異常終了などでシーンに _BAK という名前のオブジェクトが残った場合は、パネル内の 「エクスポート残骸データを手動クリア」 ボタンを押してください。

This addon exports only "Visible Bones" (based on Bone Collection visibility) to FBX when exporting from Blender. 🐾
It helps optimize avatars for VRChat and lightweight data migration to Unity.

🎮 How to Use

Install

  • Select Edit → Preferences → Add-ons → Install and choose the .py file to enable it.

Open Sidebar

  • Press N key in the 3D Viewport and open the "FBX Output" tab.

Check Settings

  • Ensure that "Export Visible Bones Only" is checked.

Export

  • Click the Open Export UI button, specify the save location like a normal FBX export, and finish.

⚠️ Notes & Specifications

【Non-Destructive Processing】

Normally restored to the original state, but residue may remain in case of abnormal termination.

  • Parent-Child Relationships: If a parent bone is hidden and a child bone is visible, the child bone is kept but the relationship might be broken.
  • FBX Structure Changes: Since the number of bones changes, always check the weights and movements on the Unity side.

🛠️ Troubleshooting

If leftover data remains

In the unlikely event of an abnormal termination leaving an object named _BAK in the scene, please click the "Manually Clear Export Residue" button in the panel.

본 애드온은 블렌더에서 FBX를 내보낼 때 "표시되어 있는 본 (Bone Collection 표시 설정)"만 추출하여 출력하는 툴입니다. 🐾
VRChat용 아바타 최적화나 Unity로의 가벼운 데이터 이전에 도움이 됩니다.

🎮 사용법 (Usage)

설치

  • 블렌더의 편집 → 환경설정 → 애드온 → 설치에서 .py 파일을 선택하여 활성화합니다.

사이드바 열기

  • 3D 뷰포트에서 N키를 눌러 "FBX 출력" 탭을 엽니다.

설정 확인

  • "표시된 본만 추출하기"에 체크되어 있는지 확인합니다.

내보내기 실행

  • 내보내기 화면 열기 버튼을 클릭하고, 일반적인 FBX 내보내기와 동일하게 저장 경로를 지정하여 완료합니다.

⚠️ 주의사항 및 사양

【비파괴 처리】

보통은 원래 상태로 복원되지만, 비정상 종료 시 잔여 데이터가 남을 수 있습니다.

  • 부모-자식 관계: 부모 본이 숨겨져 있고 자식 본이 표시된 경우, 자식 본은 유지되지만 부모-자식 관계가 해제될 수 있습니다.
  • FBX 구조 변화: 본 개수가 달라지므로 Unity에서 가중치와 동작을 반드시 확인해주세요.

🛠️ 트러블슈팅

잔여 데이터가 남은 경우

비정상 종료 등으로 씬에 _BAK라는 이름의 오브젝트가 남은 경우, 패널 내의 "내보내기 잔여 데이터 수동 지우기" 버튼을 눌러주세요.

本插件是在从 Blender 导出 FBX 时，仅提取并输出“显示的骨骼（Bone Collection的显示设置）”的工具。🐾
有助于优化 VRChat 角色以及向 Unity 输送轻量级数据。

🎮 使用方法 (Usage)

安装

  • 在 Blender 的 编辑 → 偏好设置 → 插件 → 安装 中选择 .py 文件并启用。

打开侧边栏

  • 在 3D 视图中按 N键，打开 "FBX 输出" 选项卡。

确认设置

  • 确保勾选了“仅提取显示的骨骼”。

执行导出

  • 点击 打开导出界面 按钮，像普通 FBX 导出一样指定保存路径即可完成。

⚠️ 注意事项与规格

【非破坏性处理】

通常会恢复到原始状态，但如果发生异常终止，可能会留下残留数据。

  • 父子关系: 如果父骨骼隐藏而子骨骼显示，则保留子骨骼，但可能会解除父子关系。
  • FBX 结构变化: 由于骨骼数量发生变化，请务必在 Unity 中检查权重和动作。

🛠️ 故障排除

如果残留了临时数据

万一发生异常终止导致场景中残留名为 _BAK 的对象，请点击面板内的 “手动清除导出残留数据” 按钮。
