Visible Bone FBX Exporter

本アドオンは、BlenderからのFBX出力時に、「表示されているボーン（Bone Collectionの表示設定）」のみを抽出して出力するツールです。🐾
VRChat向けアバターの最適化や、Unityへの軽量なデータ移行に役立ちます。

■ 使い方 (Usage)

インストール

  • Blenderの 編集 → プリファレンス → アドオン → インストール から .py ファイルを選択して有効化。

サイドバーを開く

  • 3Dビューポート上で Nキー を押し、「FBX出力」タブを開きます。

設定の確認

  • 「表示ボーンのみ抽出する」にチェックが入っていることを確認します。

エクスポート実行

  • エクスポート画面を開く ボタンをクリックし、通常のFBXエクスポートと同様に保存先を指定して完了。

■ 注意事項・仕様

【非破壊処理】

エクスポート時に一時的な複製データを作成するため、元のシーンデータは書き換えられません。

  • 親子関係: 親ボーンが非表示で子ボーンが表示されている場合、子ボーンは保持されますが親子関係が解除される場合があります。
  • FBX構造の変化: ボーン数が変わるため、Unity側でのウェイトや動作を必ず確認してください。

■ トラブルシューティング

残骸データが残ってしまった場合

万が一異常終了などでシーンに _BAK という名前のオブジェクトが残った場合は、パネル内の 「エクスポート残骸データを手動クリア」 ボタンを押してください。

This addon exports only "Visible Bones" (based on Bone Collection visibility) to FBX. 🐾

■ How to Use

Install

  • Edit → Preferences → Add-ons → Install and select the .py file.

Open Sidebar

  • Press N key in 3D View and find "FBX Output" tab.

Export

  • Click Open Export UI and save your FBX.
